# Timeline

**Canonical timeline:** [github.com/NautilusOSS/lendpay-timeline](https://github.com/NautilusOSS/lendpay-timeline)

Chronological history, milestones, and dated decisions for LendPay live in that repository. This index does not duplicate or fork that narrative; it only points to it so there is a single place to edit timeline content.

When adding events that affect multiple repos (releases, deprecations, org moves), update `lendpay-timeline` first, then add cross-links here or in [data/repos.json](data/repos.json) (`related_links`) if useful for navigation.
